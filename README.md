# ustai-kz-ai-platform
AI көмегімен презентациялар, тесттер және викториналар жасауға арналған оқытушылар платформасы.
# Ustai.kz

## Жоба туралы

**Ustai.kz** — оқытушыларға арналған жасанды интеллект негізіндегі білім беру платформасы. Жоба мұғалімдер мен оқытушылардың сабаққа дайындалу процесін жеңілдетуге бағытталған.

Платформа AI көмегімен презентациялар, тесттер және викториналар құруға мүмкіндік береді, сондай-ақ барлық материалдарды бір жерде сақтауға жағдай жасайды.

---

## Негізгі мүмкіндіктер

### AI Презентация генераторы

* Сабақ тақырыбы бойынша презентация құру
* Слайд құрылымын автоматты жасау
* PDF және PPTX форматтарына экспорт

### AI Тест генераторы

* Автоматты тест сұрақтарын құру
* Бір немесе бірнеше дұрыс жауап нұсқалары
* Дайын тесттерді сақтау

### AI Викторина генераторы

* Интерактивті викториналар жасау
* Автоматты бағалау
* Нәтижелерді шығару

### Жеке кабинет

* Материалдарды сақтау
* Генерациялар тарихын қарау
* Пайдаланушы профилін басқару

### Қазақ тілін қолдау

* Интерфейс қазақ тілінде
* AI генерациялары қазақ тілінде жұмыс істейді

## Орнату

1. Репозиторийді жүктеп алыңыз:

```bash
git clone https://github.com/username/ustai-kz.git
```

2. MySQL дерекқорын құрыңыз.

3. `.env` файлына Gemini API кілтін енгізіңіз.

```env
GEMINI_API_KEY=your_api_key
```

4. Веб-серверді іске қосыңыз.
